# 简介

# AuroraNative

基于 go-cqhttp (opens new window) 与 OneBot (opens new window) 标准所实现的一个 C# 开发框架，由于某些原因部分功能暂无法使用。如快速操作等。

并且本框架可以支持.NET 5/.NET Framework 4.6.1以上使用，使用.NET 5可以快捷开发一个跨平台的机器人。

注意

由于go-cqhttp 项目遭到不可抗力因素，本项目现已停止更新并归档，详情请看 —> https://github.com/Mrs4s/go-cqhttp/issues/2471

# go-cqhttp - 来自官方文档

使用 mirai (opens new window) 以及 MiraiGo (opens new window) 开发的 cqhttp golang 原生实现, 并在 cqhttp 原版 (opens new window) 的基础上做了部分修改和拓展。 文档目前还在撰写中。

测试版可前往 go-cqhttp Release (opens new window) 下载。
