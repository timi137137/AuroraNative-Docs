# 通知事件

继承基础类后，通知事件中所有事件的某些值将转为固定，并添加一些值，见下表:

字段名 数据类型 可能的值 说明
NoticeType string - 通知类型
UserID long - 发送者QQ号

# 群文件上传

方法名: GroupUpload

事件数据

字段名 数据类型 可能的值 说明
GroupID long - 群号
File File - 文件信息

# 群管理员变动

方法名: GroupManageChange

事件数据

字段名 数据类型 可能的值 说明
GroupID long - 群号
SubType string set,unset 事件子类型, 分别表示设置和取消管理员

# 群成员名片更新(核验)

方法名: GroupCardVerify

注意

此事件不保证时效性, 仅在收到消息时校验卡片

事件数据

字段名 数据类型 可能的值 说明
GroupID long - 群号
CardNew string - 新名片
CardOld string - 旧名片
